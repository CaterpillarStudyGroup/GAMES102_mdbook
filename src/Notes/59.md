#　换个方式       
\\(\partial \\) 本身一维，考虑到半移，再从一维， 
隐层的\\(l\\)是指基函数线性组合后整体增加一个平移。   
在这里， std gauss 相当于激活函数。    
连接线上的数值 \\((a_i,b_i,\omega _i)\\)是网络参数。   
\\(n\\)对应网络隐层的结点个数，需要手调。   

# RBF 神经网络   
RBF 神经网络的问题是，关于 \\(a,b\\) 的导数难求，高阶且非凸，
难以优化。只能找局部最小，因此初值很重要。
