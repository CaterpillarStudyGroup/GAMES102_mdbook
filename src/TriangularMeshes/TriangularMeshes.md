# 绘制：离散表达     

* 曲线的绘制：    
• GDI/OpenGL 绘制基本单元：点、线段      
• 曲线须离散成多边形      
* 曲面的绘制：     
• OpenGL 绘制基本单元：点、线、三角形      
• 曲面须离散成**三角形网格**
