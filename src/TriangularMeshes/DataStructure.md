# General Used Mesh Files    

* General used mesh files     
• Wavefront OBJ ( *. obj)     
• 3D Max ( *. max, *. 3ds)     
• VRML ( *. vrl)     
• Inventor (  *.  iv)      
• PLY ( *. ply, *. ply2)  
•  Py mesh lab  
• User‐defined ( *. m, *. liu)     
* Storage     
• Text – (Recommended)     
• Binary     



## Wavefront OBJ File Format    

* Vertices     
• Start with char ‘v’    
• (x,y,z) coordinates     
* Faces     
• Start with char ‘f’     
• Indices of its vertices in the file     
* Other properties     
• Normal, texture coordinates, material, etc.
