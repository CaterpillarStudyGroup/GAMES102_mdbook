# 回顾：三角网格曲面   


![](../assets/微分1-1.png)   

* 观点 1：曲面的离散逼近    
• 采样：顶点为从曲面上的采样点     
• 构网：每个三角面为线性平面     
• 本质：分片线性逼近    

![](../assets/微分2.png)   

* 观点 2：平面图的嵌入     
• 平面图    
• 图的顶点提升 (lifting) 至三维空间   
• 本质：二维流形    

![](../assets/微分3.png)
