# Gauss函数

> &#x2705; 一维 RBF 称为 Gauss 函数

$$
g_{\mu ,\sigma } = \frac{1}{\sqrt{2\pi } } e^{-\frac{(x-\mu )^{2} }{2\sigma ^{2} } } 
$$

均值 \\(\mu\\) 方差\\(\sigma\\)  

**不同µ和0的 Gauss 函数都线性无关.**
