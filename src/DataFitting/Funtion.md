# 为什么用多项式？

> 多项式逼近的优点
> 1. 易于计算。 先滑
> 2. 表达能力足够。Pn(x) 可以在一定误差内拟合任意f(x)

# 用Bernstein多项式做逼近

$$
b_{n,j} = \binom{n}{j} x^j (1-x)^{n-j}
$$

# 关于Bernstein函数…

$$
B_n(f,x) = \sum f\left ( \frac{i}{n}  \right ) b_{n,j}\left ( x \right )  
$$

> &#x2705; f是采样点，b可以看作是f的组合系数
